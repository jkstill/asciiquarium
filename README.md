asciiquarium
============

Based on the original Asciiquarium by Kirk Baucom, this is a fork of the Android live wallpaper "Aquarium Live Wallpaper" by Claudio Matsuoka. 

See the original Asciiquarium at http://www.robobunny.com/projects/asciiquarium/html/ for more information.

See the original "README" file that is included here, for more information.

Changes:

- added a jellyfish
- added a scubadiver

Added three new keys:

- f: faster
- s: slower
- d: default speed
